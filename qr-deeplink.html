Deep Link QR Code Generator Documentation

This document provides an overview and usage instructions for the Advanced Deep Link QR Code Generator tool.

Overview

The Deep Link QR Code Generator is a client-side web utility designed to simplify the creation and testing of game deep links. It allows developers and QA testers to dynamically build complex URLs with various parameters, generate a scannable QR code, and launch the game on a mobile device with the specified configuration. The tool is fully client-side and uses local storage to remember your settings, making it efficient for repeated use.

Key Features

  • Dynamic URL Generation: Automatically builds the deep link as you change parameters.
  • Environment Selection: Easily switch between Trunk, Trunk Liveops, QA3, or provide a Custom Backend URL.
  • PR Number Support: Automatically formats the backend host with a PR number for applicable environments.
  • Extensive Cheat Parameters: A comprehensive list of optional query parameters can be enabled and configured.
  • Live QR Code: The QR code updates in real-time as you modify the deep link.
  • Sharable Links: Generate a unique URL that contains all your current settings for sharing pre-configured deep links with team members.
  • Server Alive Check: Automatically checks if the selected backend server is reachable.
  • Grafana Log Integration: Provides a direct link to Grafana logs for specific environments.
  • Local Storage Persistence: Remembers your last-used configuration for convenience.

How to Use

Base Configuration

  1. Select an Environment: Choose a predefined environment from the dropdown.
  2. PR Number: For environments like `Trunk` or `QA3`, you can enter a PR number to construct the hostname.
  3. Custom Backend URL: If you select "Custom", provide the full hostname for your target server.

Additional Query Parameters

Enable and configure optional parameters ("cheats") from the list provided.

  • To include a parameter, check the box next to its name.
  • Once enabled, set its value in the corresponding input field.
  • The deep link and QR code will update instantly.

Alive Check

After you stop typing in a relevant field, the tool performs a reachability check on the configured server.

Note on CORS: The alive check uses a no-cors request. This means it can only verify network reachability. It cannot determine if the response was a "200 OK" or a "404 Not Found" due to browser security policies (CORS). A "Reachable" status confirms the server is accessible from your network.

Actions & Buttons

  • Start Game: On a mobile device, this button will attempt to open the generated deep link.
  • Copy Sharable Link: Copies a link to the tool with all current settings preserved.
  • Reset All Settings: Clears all saved settings and resets the form to its default state.

File Structure

The tool is composed of the following files:

  • qr-deeplink.html - The main HTML structure
  • style.css - All the CSS styling
  • script.js - All the JavaScript logic
  • grafana-icon.svg - The icon for the Grafana link
