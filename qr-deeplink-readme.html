Deep Link QR Code Generator Documentation

This document provides an overview and usage instructions for the Advanced Deep Link QR Code Generator tool. Read it. Or don't. I'm a document, not a cop.

Overview

The Deep Link QR Code Generator is a client-side web utility designed to simplify the creation and testing of game deep links. It was created because typing long URLs is a terrible way to spend your finite time on this planet. It allows developers and QA testers to dynamically build complex URLs, generate a scannable QR code, and launch the game on a mobile device. The tool is fully client-side and uses local storage to remember your settings, so your countless cheat parameters are safe... until you clear your cache.

Key Features

  • Dynamic URL Generation: Automatically builds the deep link as you change parameters, because who has time to press a "generate" button anymore?
  • Environment Selection: Easily switch between Trunk, Trunk Liveops, QA3, or provide a Custom Backend URL for when you're feeling adventurous.
  • PR Number Support: Automatically formats the backend host with a PR number. Magic!
  • Extensive Cheat Parameters: A comprehensive list of optional query parameters you can enable. Use your power wisely.
  • Live QR Code: The QR code updates in real-time. Don't stare at it too long, it might stare back.
  • Sharable Links: Generate a unique URL that contains all your current settings. Perfect for annoying your colleagues with your very specific testing setup.
  • Server Alive Check: Automatically pings the server to see if it's awake. It's more responsive than some people before their morning coffee.
  • Grafana Log Integration: Provides a direct link to Grafana logs. Dive into a sea of logs and question all your life choices.
  • Local Storage Persistence: Remembers your last-used configuration. It has a better memory than you do.

How to Use

Base Configuration

  1. Select an Environment: Pick your poison from the dropdown.
  2. PR Number: For environments like Trunk or QA3, you can enter a PR number to construct the hostname.
  3. Custom Backend URL: For those who like to live dangerously. Go on, enter your secret server address. No one's looking.

Additional Query Parameters

Here's where the real "fun" begins. Check the boxes to enable parameters that will inevitably lead to finding (or creating) bugs.

  • To include a parameter, check the box. It's not rocket science.
  • Once enabled, set its value in the corresponding input field.
  • The deep link and QR code will update instantly, whether you're ready or not.

Alive Check

After you stop typing in a relevant field, the tool performs a reachability check on the configured server.

Note on CORS: The alive check uses a no-cors request. This means it can only verify that something, anything, exists at that address. It could be a web server or a very clever toaster. Due to browser security policies (CORS), we can't tell the difference. A "Reachable" status confirms the server is accessible from your network.

Actions & Buttons

  • Start Game: On a mobile device, this button will attempt to open the generated deep link. If it doesn't work, that's probably a "you" problem.
  • Copy Sharable Link: Copies a link to the tool with all current settings preserved. Share it with your favorite coworker to make their day slightly more complicated.
  • Reset All Settings: The panic button. Clears all your carefully crafted settings and returns to a state of blissful ignorance.
